Store Management System – Mobile Shop

*Project Description
This project is a Store Management System for a Mobile Shop developed using VB.NET and SQL Server Management Studio.  
The application is designed to manage daily shop operations such as product management, sales, customer details, and billing in an efficient and user-friendly manner.

*Technologies Used
- Programming Language: VB.NET  
- Framework: .NET Framework  
- Database: SQL Server  
- Database Tool: SQL Server Management Studio (SSMS)  
- IDE: Visual Studio  

*Features
- Add, update, and delete mobile products
- Manage customer information
- Handle sales and billing
- Store and retrieve data using SQL Server
- Simple and user-friendly interface

Project Structure
- `New Mobail Shop/` – Source code of the application  
- `New Mobail Shop.sln` – Visual Studio solution file  
- `SQLQuery4.sql` – SQL database queries and tables
